## CBETA 特有信息

### ID 格式
- TX{N}n{NNNN}：TX为全书标识，N为实体书册号，n{NNNN}为逻辑编号
- TX00na001 为编纂说明（独立于20编之外）
- 部分编跨多册：第一编(TX01-TX02)、第四编(TX04-TX05)、第六编(TX07-TX09)、第七编(TX10-TX15)、第十三编(TX20-TX23)、第十七编(TX26-TX27)、第十八编(TX27-TX28)、第十九编(TX29-TX31)

### 四藏分类（来自CBETA编纂凡例）
- 甲 法藏（7编）：第一至第七编
- 乙 制藏（3编）：第八至第十编
- 丙 论藏（4编）：第十一至第十四编
- 丁 杂藏（6编）：第十五至第二十编

## 网页版特有信息

### ClassID 编号体系
- 每个编/子目有独立 ClassID，范围从94到180
- 部分 ClassID 之间有间隔（如94-95-99），可能是预留或已删除条目

### 子目结构
网页版在编下还细分了「子目」（classname），如第一编下：概论(13篇)、判摄(19篇)、源流(18篇)
