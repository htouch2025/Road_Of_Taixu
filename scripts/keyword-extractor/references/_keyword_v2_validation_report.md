### 已知问题

1. **修行次第语义重叠**：该标签在知识域和功能取向两维度下均有出现。这不是标注错误——佛法導言确实既属于修行次第的知识领域，又以修行次第为其功能取向——但可能造成后续统计和分析的混淆。建议在 prompt 中增加此说明。

2. **词汇表覆盖偏向**：当前词表以教理为核心，对制度、社会文化、戒律等领域覆盖率偏低。gap 日志中「僧制」「政教分離」「進化論」「階級」等术语均属此类。

3. **座谈体信号稀疏**：对话体中学术术语密度低，Phase A 候选数仅 79，影响 Phase B 的候选支撑。

4. **超长文效率**：54K 文章全文通读成本过高，当前 TOC + 候选摘要的近似方法可行，但存在遗漏篇章结构层面知识域信号的风险。
