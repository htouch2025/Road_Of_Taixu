# 太虚大师全书 CBETA Catalog

Complete catalog of the 太虚大师全书 as organized in CBETA, with CBETA IDs and 四藏 classification.

## ID Format

```
TX{NN}n{NNNN}
```
- `TX` = 太虚大师全书 prefix
- `{NN}` = 实体书册号 (01–32)
- `n{NNNN}` = 逻辑编号 (na001 = 编纂说明, n0001–n0020 = 第1–20编)

## 编名说明

- 第十一至十三编的全名在 CBETA 中为「宗依論」「宗體論」「宗用論」，但纸质书使用「真现实論宗依論」「真现实論宗體論」「真现实論宗用論」。本项目使用后者（加入「真现实論」前缀）。
- 编纂说明 (TX00na001) 不属于任何编。
