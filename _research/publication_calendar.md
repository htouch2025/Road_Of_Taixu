## 说明

本文档是太虚大师全书刊载信息全录v2中所有出现过的刊物的卷号/期号与公历年月对照表。
用于将刊载出处（如「见海刊九卷第九期」）中的卷期引用解析为具体日期。

### 分类说明

| 类别 | 说明 | 对照表作用 |
|------|------|-----------|
| **卷期制期刊** | 引用格式为X卷Y期 | 需建立 卷→年、期→月 映射 |
| **期号制刊物** | 仅有期号无卷号 | 需建立 期号→年月 映射 |
| **日期嵌入型** | 引用中直接带日期 | 无需额外对照表 |
| **报纸** | 日报，引用带日期 | 无需对照表 |
| **单行本/丛书** | 书籍/讲演集/出版社 | 无期号概念 |
| **出版机构** | 出版社/书局 | 非刊物 |

### 换算公式

- **海潮音**：`卷N ≈ 公元1919+N年`，`期M ≈ M月`
- **通用**：若每年一卷，则 `卷N = 创刊年 + N - 1`
- **中国数码**：`一=1, 二=2, ... 十=10, 廿=20, 卅=30`
- **合刊**：取第一个期号的月份

### 佛历换算（参考）

太虚全书中出现的佛历年有不同换算标准，常见两种：
- 佛历 = 公元 + 544（标准南传佛教历）
- 佛历 = 公元 + 989～1027（某些中国佛教文献使用的偏移）

实际使用时，建议以文本上下文和太虚生平（1889-1947）为约束，取落在合理范围内的年份。
